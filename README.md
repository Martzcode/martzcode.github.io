# 🚀 Portfolio de Marcello Samiandrisoa

Ce dépôt contient le code source de mon portfolio professionnel. C'est un site web moderne, performant et entièrement responsive conçu pour présenter mes compétences et mes expériences en tant que Développeur Fullstack.

## 🛠️ Stack Technique

- **HTML5** : Structure sémantique et optimisée SEO.
- **Tailwind CSS (CDN)** : Framework CSS utilitaire pour un design premium et rapide.
- **JavaScript (Vanilla)** : Logique personnalisée pour la navigation, le mode sombre et les animations.
- **Google Fonts** : Utilisation de *Montserrat* (titres) et *Lato* (corps).
- **FontAwesome & Devicons** : Iconographie moderne.

## 📁 Architecture du Projet

Le projet est organisé de manière modulaire pour faciliter l'ajout de futurs contenus :

- `index.html` : La page d'accueil principale optimisée pour le SEO.
- `style.css` : Styles globaux, variables de thème et système de défilement One-Page.
- `script.js` : Gestion des interactions (thème, navigation mobile, carrousel).
- `/assets` : Ressources statiques du site.
  - `/icons` : Icônes SVG personnalisées (dont le terminal moderne).
  - `/images` : Captures d'écran et médias.
  - `/logos` : Logos des entreprises et universités.
  - `/cursors` : Curseurs personnalisés inspirés de macOS.
  - `/docs` : Documents téléchargeables (CV, etc.).
- `/site` : Contenus additionnels et futures pages.
  - `/projects` : Pages de détails des projets.
  - `/pages` : Pages annexes.
  - `/components` : Éléments HTML réutilisables.

## 🌟 Fonctionnalités Clés

- **Mode Sombre** : Détection automatique et bascule manuelle avec transition douce.
- **One Page Scroll** : Système de défilement vertical par section avec "snap".
- **Carrousel d'Expériences** : Navigation horizontale avec indicateurs interactifs.
- **Optimisation SEO** : Meta-tags complets et données structurées JSON-LD.
- **Adaptivité** : Optimisé pour tous les écrans (mobile, tablette, desktop).
